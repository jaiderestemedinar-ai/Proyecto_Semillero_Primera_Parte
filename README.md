# 🚗 Carro Omnidireccional y Brazo Robótico

## Semillero fafafa - Jaider Medina 💪

Este repositorio documenta el desarrollo de un vehículo omnidireccional y la integración de un brazo robótico. El sistema está diseñado para tener una conexión y control independientes.

### ⚙️ Hardware Principal
* **Microcontrolador Central:** Pic18F45k22.
* **Comunicación Inalámbrica:** Módulos XBee y ESP8266.
* **Tracción:** Ruedas Mecanum para lograr el desplazamiento tipo "cangrejo".

### 💻 Software y Simulaciones
* Diseño CAD y simulaciones de cinemática realizadas en SolidWorks y CoppeliaSim.
